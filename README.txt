=======================================================================================================================================
                                               * Algorithm for Identical Binary Trees *
=======================================================================================================================================

1) Read both trees from command line
2) Insert entered value into tree level by level from left to right (Complete binary tree)
3) Check if both the trees have equal height, if yes check if the value at the node are equal
   and recursively check if both left and right subtrees are identical.

======================================================================================================================================
                                                       * Assumptions *
======================================================================================================================================

1) The values at a node is an integer.

======================================================================================================================================

This is a java program. Use Java 8 library, import this project and run main class (IdenticalTrees).
Enter input as prompted in command line.
PS : Please enter only integer values for the nodes.
